Place your music files inside a forlder named "Music" inside the game's directory,
place dsound.dll and dsound_ori.dll also into the games directory

now, you should hear your music in the (Evil Nine - Happy Ending) track.
also, you can hear the original (Evil Nine - Happy Ending) track using the force stop key

the music must have the following settings (convert with audacity or ffmpeg)
16-bit Signed PCM Little endian
2 channels
44100 sample rate
Raw (header-less)
English file name and path

Controlls (now working inside the game)
Ins: force enable user music (disables the game music)
media Play/pause: play/pause
media stop: disable force user music
Next song: go to a next (random) song
